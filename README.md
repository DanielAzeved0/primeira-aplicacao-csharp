# Primeira Aplicação em C#

Um projeto inicial de aprendizado em C# desenvolvido como parte do curso de formação em desenvolvimento .NET.

## 📋 Descrição do Projeto

Este projeto representa o primeiro passo no aprendizado de C# e .NET. Ele foi criado com o objetivo de consolidar conceitos fundamentais da linguagem, como:

- Estrutura básica de uma aplicação C#
- Sintaxe e tipos de dados
- Entrada e saída de dados
- Conceitos iniciais de programação orientada a objetos

## 🎓 Sobre o Curso

Este projeto foi desenvolvido como parte do curso:

**"C#: Criando sua primeira aplicação"**

- 📘 [Link do Curso](https://cursos.alura.com.br/course/csharp-criando-primeira-aplicacao)
- 🏫 Plataforma: Alura
- 🎯 Objetivo: Introdução ao C# e criação da primeira aplicação

O curso aborda os fundamentos necessários para começar a desenvolver aplicações em C# usando a plataforma .NET.

## 🛠️ Tecnologias Utilizadas

- **Linguagem**: C# 
- **Framework**: .NET 10.0
- **IDE**: Visual Studio

## 📁 Estrutura do Projeto

```
PrimeiroProjeto/
├── Program.cs           # Arquivo principal da aplicação
├── PrimeiroProjeto.csproj  # Configuração do projeto
└── bin/                 # Arquivos compilados (gerados)
```

## 🚀 Como Executar

1. Clone ou baixe o repositório
2. Navegue até a pasta do projeto:
   ```bash
   cd PrimeiroProjeto
   ```
3. Compile e execute:
   ```bash
   dotnet run
   ```

## 📝 Requisitos

- .NET 10.0 ou superior instalado
- Visual Studio ou VS Code (opcional)

## 📌 Notas

Este é um projeto educacional e serve como base para evoluir em conhecimento de C# e .NET.
